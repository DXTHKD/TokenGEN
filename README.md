# DXTHKD TokenGEN + Checker

Made this as my first project since im learning python rn.

## Installation

# Win
```
Firstly Download the repo
then
$ pip install -r requirements.txt
$ python main.py
```
# Linux
```
$ apt update
$ apt upgrade
$ apt install git
$ git clone https://github.com/DXTHKD/TokenGEN.git
$ cd TokenGEN 
$ pip install -r requirements.txt
$ python3 main.py
```

Enjoy!
